git checkout master = para mudar para a branch de testes;
git add (arquivo) = para adicionar arquivos na area de stagging;
git add -A = para adicionar TODOS os arquivos presentes no repositorio local;
git commit -m "exemplo de titulo" = para fazer um commit ;
git push origin (endereço do repo) = para ENVIAR o commit para o github;
git pull origin (branch) = para PUXAR o contéudo do github para seu repositorio local;
